# Neural Gas Clustering

Реализация алгоритма **нейронного газа (Neural Gas)** для кластеризации на Python. Алгоритм основан на конкурентном обучении и адаптивной топологии графа, что позволяет выявлять кластеры произвольной формы без предварительного задания их числа.

## Как работает алгоритм

Нейронный газ — это нейросетевой метод кластеризации, в котором нейроны (центроиды) перемещаются в пространстве признаков, подстраиваясь под распределение данных. Ключевые особенности данной реализации:

- **Динамическое добавление нейронов** — при достижении заданного числа итераций добавляется новый нейрон между двумя нейронами с наибольшей локальной ошибкой.
- **Управление связями** — между нейронами поддерживаются связи (рёбра), которые стареют и удаляются, если превышают `max_age`. Это позволяет формировать топологию графа, отражающую структуру данных.
- **Итоговая кластеризация** — после обучения кластеры определяются как компоненты связности полученного графа.

## Параметры функции `Neural_gas`

| Параметр    | Значение по умолчанию | Описание |
|-------------|----------------------|-----------|
| `norm_coeff` | $10^5$               | Коэффициент нормализации входных данных (все признаки делятся на него) |
| `epsilon1`   | 0.2                  | Скорость обучения для победившего нейрона |
| `epsilon2`   | 0.005                | Скорость обучения для соседей победителя |
| `max_age`    | 25                   | Максимальный возраст ребра; связи старше этого значения удаляются |
| `epochs`     | 35                   | Количество эпох (проходов по всем данным) |
| `count_iter` | 80                   | Частота добавления новых нейронов (каждые `count_iter` итераций) |
| `err_alpha`  | 0.5                  | Коэффициент уменьшения локальной ошибки после вставки нового нейрона |
| `err_betta`  | 0.954                | Коэффициент наследования локальной ошибки новым нейроном |

## Примеры из ноутбука
В ноутбуке приведены три тестовых сценария:

- **Простые фигуры, явная кластеризация** — три группы точек, сгенерированных в разных областях. Алгоритм успешно разделяет их.

- **Неявная кластеризация** — данные с перекрывающимися кластерами. Алгоритм справляется за счёт адаптивной топологии.

- **Сложные фигуры** — кольцо и синусоидальная полоса. Нейронный газ выделяет кластеры, соответствующие форме данных.

## Примечания

- Алгоритм не требует задания числа кластеров — оно определяется автоматически по компонентам связности.

- Для улучшения стабильности рекомендуется нормализовать данные (коэффициент norm_coeff).

- Параметры epsilon1, epsilon2, max_age, count_iter влияют на скорость сходимости и топологию.
